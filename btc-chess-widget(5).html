About This Widget

🎯 What Is This?

This widget displays Freestyle (Chess960) starting positions derived from Bitcoin block hashes. It combines cryptographic randomness from the Bitcoin blockchain with the creativity of Chess960.

⚙️ How It Works

Live Mode:

  • Fetches the latest Bitcoin block hash from the blockchain
  • Converts the hash to a number using hash % 960
  • Generates the corresponding Chess960 position (0-959)
  • Auto-updates every 30 seconds to check for new blocks

Manual Override Modes:

  • Block Height: Enter a block number (e.g., 870000) to fetch its hash from the blockchain
  • Text/Data: Enter any text or string. All strings (including block hashes) are converted to numbers using character codes, then mod 960 is applied to get the Chess960 position.
  • Position #: Direct entry of any Chess960 position (0-959)

🔐 The Bitcoin Connection

What is a Block Hash?

A Bitcoin block hash is the double SHA-256 cryptographic hash of the block header. The header contains:

  • Version number
  • Previous block's hash (linking blocks together)
  • Merkle root (hash of all transactions in the block)
  • Timestamp
  • Difficulty target
  • Nonce (random number miners adjust)

Miners compute billions of hashes per second, changing the nonce each time, until they find a hash that meets Bitcoin's difficulty requirement (starts with enough zeros). This hash is:

  • Unpredictable: Can't be known before mining completes
  • Immutable: Can't be changed without re-mining
  • Unique: Each block has exactly one valid hash
  • Public: Transparent and verifiable by anyone

This makes block hashes perfect for generating provably fair Chess960 positions. The position number is calculated by converting the hash string to a number using character codes, then applying mod 960.

♟️ Chess960 Explained

Chess960 (also called Fischer Random Chess) has 960 possible starting positions following these rules:

  • Bishops must be on opposite-colored squares
  • King must be between the two rooks
  • Position 518 is standard chess
